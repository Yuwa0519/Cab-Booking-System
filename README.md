# Cab-Booking-System
In particular, this Online Cab Booking System project focuses mainly on booking cabs and their management. To be more precise, the system helps clients filter, and book cabs online. Also, the system contains all the client, and cab management features with it. In addition, the system allows managing client bookings, website settings, and more. Evidently, this project contains an admin panel with a customer and driver panel. In an overview of this web application, customers can simply register on the website in order to book cabs.
